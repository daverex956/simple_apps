# simple_apps
This repository contains scripts for listing and running apps.

### Install Dependencies
```bash
pip install gTTS
pip install tqdm
pip install pdfplumber
pip install pytesseract
pip install pillow
pip install ffmpeg-python
pip install moviepy
pip install pyfiglet
pip install termcolor
pip install openai

brew install ffmpeg
```

### Run
```bash
python3 run.py
```

### Terminal Example
```bash
User % python3 run.py

--- Choose App --- 
1. PDF_Reader
2. Text_to_Speach
3. Add_Video_Outro
4. Remove_Metadata
5. BTC_Price
6. Exit

```
